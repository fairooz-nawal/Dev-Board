Question-1: What are the different ways to select an element in the DOM?


An element can be selected using the following ways

  • Using getElementById.
  • Using getElementsByClassName.
  • Using getElementsByTagName.
  • Using querySelector.
  • Using querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?


"InnerHTML" is the complete HTML content within an element, including tags, "innerText" is the visible text content of an element, ignoring HTML tags and hidden text and "textContent" is all text content within an element, including hidden text, but without considering HTML tags or styling, preserving whitespace and line breaks

Question-3: What is event delegation in the DOM?


Event delegation means the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.

Question-4: What is event bubbling in the DOM?


Event bubbling is a type of DOM event propagation where the event first triggers on the innermost target element, and then successively triggers on the ancestors (parents) of the target element

Question-5: How do you create, add, and remove elements using JavaScript?


To add, remove, and modify HTML elements using JavaScript, you can use methods like createElement , appendChild , removeChild , and properties like innerHTML and textContent
